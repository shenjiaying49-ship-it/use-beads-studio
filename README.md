# 自动DIY拼豆图纸

一个浏览器内运行的拼豆图纸生成器。
是否总是面对着看到了心仪的图纸，但是一个拼图板拼不下的困扰？  
是否面临着下班之后想拼会儿豆放松一下，但拼的时间太长了又觉得劳累的问题？  
不知道把拼豆储存在哪里？ 害怕下一次拼豆拿出来就会容易损耗，想要一次性享受拼+烫的快乐又不想花太多时间？
这是你能找到的最轻量最好用的产品

在线地址：

`https://shenjiaying49-ship-it.github.io/use-beads-studio/`

## 功能

- 上传图片并生成编号网格
- 白色背景不计入拼豆颗数
- 上传后先按基准最小尺寸等比缩小
- 在基准尺寸上继续缩小，减少颗数和细节
- 导出当前页图纸 PNG

## 本地打开

在项目目录运行：

```bash
python3 -m http.server 8000
```

然后访问：

`http://127.0.0.1:8000/`
